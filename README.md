# Teach Journey :open_book:
Este juego utiliza las definiciones primarias de Teoría de conjuntos y la representación de fracciones para poder crear un juego con un objetivo educativo y divertido. Este programa se centra en la recolección y depósito de recursos, mientras se esquiva o ataca a enemigos que intentarán impedir esta tarea.
   
[![Teach Journey](https://img.youtube.com/vi/o-0iuWVV_Ow/hqdefault.jpg)](https://youtu.be/o-0iuWVV_Ow)

## Cómo jugar :video_game:
- Al iniciar el nivel, podrás visualizar recursos en el suelo. Al tocarlos, podrás reconocerlo mediante imágenes y, relacionándolos con las imágenes de los contenedores, podrás depositarlos. En caso de error, se descontará un punto. 
- Tienes que depositar la cantidad indicada mediante fracciones en la parte lateral derecha. Si se excede el número, se descontará la puntuación. Por cada uno que completes de los 3, se te otorgarán 250 puntos, que junto al tiempo serán parte de tu puntaje final.
- Mientras realizas esto, habrá enemigos que te perseguirán y solo contarás con una bala para atacarlos. También, tienes poderes de un solo uso, por lo que deberás aprovechar su uso.
- Hay 3 niveles y en cada uno los conjuntos y fracciones aumentarán su dificultad. Por ejemplo, hay elementos que pertenecerán a más de un conjunto y algunos dan más ítems al colocarlos en los contenedores. 
- Al final, podrás visualizar un ranking en el que aparecen los 3 mejores jugadores de cada nivel.  
## Avances técnicos :gear:
- Estructura base de nivel para poder crearlos con facilidad.
- Recolección de recursos y generación de enemigos.
- Base para la creación e interacción de dibujos 2d.
- Ranking y animación para cada nivel.
- Sistema de fracciones y verificación de elementos para los conjuntos.

## Limitaciones :wrench:
- Uso básico de structs
- Arreglos con punteros.
- Uso de una sola biblioteca.
